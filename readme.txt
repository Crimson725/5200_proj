# README for Group 9's Final Project

## Project Title
Exploring the Impact of Internet Accessibility on Economic Growth

## Thesis
This project investigates the hypothesis that greater access to the internet leads to higher economic growth.

## Project Structure
The project directory contains three key files:
- `5200_proj.ipynb`: This Jupyter notebook contains all the analysis and visualizations pertinent to our project.
- `GDP_growth.csv`: This dataset includes the annual GDP growth rate for various countries.
- `Individuals_using_internet.csv`: This dataset captures the percentage of individuals using the internet in different countries.

## Running the Notebook
To run the analysis and view the results, please follow these steps:

1. Ensure you have Jupyter Notebook installed.
2. Navigate to the project directory containing the `5200_proj.ipynb` file.
3. Run the notebook and execute all cells.
